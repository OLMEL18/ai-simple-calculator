# Calculator Web App

Простий односторінковий вебзастосунок-калькулятор.

## Як запустити локально

Відкрийте `index.html` у браузері.

## Як опублікувати

### GitHub Pages
1. Створіть новий репозиторій, наприклад `calculator-webapp`.
2. Завантажте `index.html` у корінь репозиторію.
3. Відкрийте **Settings → Pages**.
4. Виберіть **Deploy from a branch**.
5. Branch: `main`, folder: `/root`.
6. Збережіть і дочекайтесь посилання.

### Netlify Drop
1. Відкрийте Netlify Drop.
2. Перетягніть папку `calculator-artifact`.
3. Отримайте публічне посилання.

### Plesk / звичайний хостинг
Завантажте `index.html` у потрібну папку сайту.
